# Code of Conduct

## Our Pledge
We pledge to make participation in our project harassment-free.

## Our Standards
Examples of behavior that contributes to a positive environment include respectful communication.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the standards.

## Scope
This Code of Conduct applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported to the project team.

## Enforcement Guidelines
Project maintainers will follow appropriate guidelines when handling violations.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
